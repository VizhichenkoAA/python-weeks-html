# LLM rules — week 14 (variant 03)

## Можно спрашивать у LLM
- Объяснить тренд по **уже рассчитанным** метрикам
- Сформулировать выводы, риски, гипотезы
- Предложить следующие проверки данных

## Нельзя без проверки кодом
- Считать max/min/mean/sum/count
- Искать top-N
- Считать корреляции по raw

## Паттерн
1. Код считает агрегаты (`llm_summary.py`, SQL, pandas)
2. LLM получает 3–5 строк контекста
3. Результат в `docs/llm/summary.md`
4. Запись в `docs/LLM_Usage_Log.md`

Запуск:

```bat
scripts\run_llm_summary.bat
```

API-ключ только в `.env`, не в git.
