# Data Dictionary — mart_variant_03 (вариант 03)

Человеко-ориентированное описание колонок витрины `mart_daily` / таблицы `mart_variant_03`.

| Колонка | Бизнес-смысл | Единица | Пример / примечание |
|---|---|---|---|
| city_id | Идентификатор города | — | `RU_NSK` |
| date | Календарный день наблюдений | date (локальная TZ) | `2025-01-01` |
| T_mean | Средняя температура за день | °C | Среднее по hourly `temperature_2m` |
| P_sum | Сумма осадков за день | мм | Сумма hourly `precipitation` |
| wind_max | Максимальная скорость ветра за день | км/ч | Max hourly `wind_speed_10m` |
| rainy_hours | Число часов с осадками > 0 | час | Счётчик по hourly |
| obs_hours | Число часовых строк в дне | час | Ожидается до 24 |
| city_name | Название города | текст | Из `reference/cities.csv` |
| country_code | Код страны | ISO-2 | `RU` |
| latitude | Широта | градусы | 55.0084 |
| longitude | Долгота | градусы | 82.9357 |
| timezone | Часовой пояс агрегации | IANA | `Asia/Novosibirsk` |
| wind_rank | Ранг дня по ветру внутри города | rank | 1 = самый ветреный день |

## Business key
`(city_id, date)` — одна строка на город и день.

## Единицы (governance)
- Температура: **°C** (Open-Meteo)
- Осадки: **мм**
- Ветер: **км/ч** (единицы API Open-Meteo)

Ошибка конвертации единиц без фиксации в контракте — см. `broken_units.py`.
